# flp
Get fullpath of file or working directory

## How to Install
### cargo build
```
git clone https://github.com/Q0tzly/flp.git"
cd flp
cargo build --release
cp target/release/flp /usr/local/bin
cd .. && rm -rf flp
```

## Usage
  flp -h, --help     put help
  flp -v, --version  put version
  flp                put working dir
  flp <PATH>         put full path of file or dir
